# Version 1 (MVP)

## Goal

Build a simple OCR Intelligence Platform that helps users choose the right OCR service based on their document purpose and generates a clean Markdown file.

---

## User Flow

### Step 1: Select Purpose

* Research & Knowledge Extraction
* Legal Documents
* Financial Documents
* Invoices & Receipts
* Forms
* Scanned Documents
* Handwritten Documents

### Step 2: Upload Document

Supported Formats:

* PDF
* DOCX
* TXT
* PNG
* JPG
* JPEG
* TIFF

### Step 3: OCR Recommendation

Show recommended OCR providers based on selected purpose.

Example:

Research Documents:

* Mistral OCR
* LlamaParse

Invoices:

* Amazon Textract
* Google Document AI

### Step 4: OCR Processing

Extract:

* Text
* Tables
* Metadata

### Step 5: Generate Markdown

Output:

* output.md

---

## Features

### Included

* Purpose Selection
* File Upload
* OCR Provider Selection
* OCR Processing
* Markdown Export

### Excluded

* OCR Benchmarking
* Q&A
* Analytics
* Team Collaboration
* Batch Processing

---

## Success Criteria

* Upload document
* Select purpose
* Select OCR
* Generate Markdown
* Download output
